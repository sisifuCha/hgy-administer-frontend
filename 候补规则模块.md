## GET 查看候补规则

GET /admin/getWaitingRules

将数据库里的候补表的条目全都展示出来

### 请求参数

|名称|位置|类型|必选|说明|
|---|---|---|---|---|
|Authorization|header|string| 否 |none|

> 返回示例

> 200 Response

```json
{
  "code": 0,
  "msg": "string",
  "data": {
    "rules": [
      {
        "id": 0,
        "rule_name": "string",
        "rule_value": 0,
        "description": "string",
        "created_at": "string",
        "updated_at": "string"
      }
    ]
  }
}
```

### 返回结果

|状态码|状态码含义|说明|数据模型|
|---|---|---|---|
|200|[OK](https://tools.ietf.org/html/rfc7231#section-6.3.1)|none|Inline|

## PUT 更改挂号规则表的值

PUT /admin/changeRuleValue

根据newVlues里的各个候补规则id-候补规则值对应关系，修改候补规则表里的对应元组数据

> Body 请求参数

```json
{
  "newValues": {
    "1": 95,
    "2": 88,
    "3": 59
  }
}
```

### 请求参数

|名称|位置|类型|必选|说明|
|---|---|---|---|---|
|Authorization|header|string| 否 |none|
|body|body|object| 是 |none|
|» newValues|body|object| 是 |none|
|»» 1|body|integer| 是 |id为1的候补规则的值|
|»» 2|body|integer| 是 |id为2的候补规则的值|
|»» 3|body|integer| 是 |id为3的候补规则的值|

> 返回示例

> 200 Response

```json
{}
```

### 返回结果

|状态码|状态码含义|说明|数据模型|
|---|---|---|---|
|200|[OK](https://tools.ietf.org/html/rfc7231#section-6.3.1)|none|Inline|
