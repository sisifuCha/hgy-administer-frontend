# 管理端/患者信息管理模块

## POST 分页获取患者表

POST /admin/getPatients

> Body 请求参数

```json
{
  "pageNum": 1,
  "pageSize": 20
}
```

### 请求参数

|名称|位置|类型|必选|说明|
|---|---|---|---|---|
|Authorization|header|string| 否 |none|
|body|body|object| 是 |none|
|» pageNum|body|integer| 是 |表示第几页|
|» pageSize|body|integer| 是 |表示一页多少条条目|

> 返回示例

> 200 Response

```json
{
  "code": 0,
  "msg": "string",
  "data": {
    "List": [
      {
        "id": "string",
        "birth": "string",
        "id_num": "string",
        "medical_insuranceid": "string",
        "reimburse_id": "string",
        "status": "string"
      }
    ]
  }
}
```

### 返回结果

|状态码|状态码含义|说明|数据模型|
|---|---|---|---|
|200|[OK](https://tools.ietf.org/html/rfc7231#section-6.3.1)|none|Inline|

### 返回数据结构

状态码 **200**

|名称|类型|必选|约束|中文名|说明|
|---|---|---|---|---|---|
|» code|integer|true|none||none|
|» msg|string|true|none||none|
|» data|object|false|none||none|
|»» List|[[patient](#schemapatient)]|true|none||none|
|»»» id|string|true|none||none|
|»»» birth|string|true|none||none|
|»»» id_num|string|true|none||身份证号|
|»»» medical_insuranceid|string|true|none||none|
|»»» reimburse_id|string|true|none||none|
|»»» status|string|true|none||none|

## GET 查看某个患者的详细信息

GET /admin/getPatient

### 请求参数

|名称|位置|类型|必选|说明|
|---|---|---|---|---|
|id|query|string| 否 |患者id|
|Authorization|header|string| 否 |none|

> 返回示例

> 200 Response

```json
{
  "code": 0,
  "msg": "string",
  "data": {
    "id": "string",
    "birth": "string",
    "id_num": "string",
    "medical_insuranceid": "string",
    "reimburse_id": "string",
    "status": "string",
    "email": "string",
    "pass": "string",
    "name": "string",
```
