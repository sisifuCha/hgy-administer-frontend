## POST 医生班次变更申请 

POST /admin/schedule_change_request

提交班次变更申请
输入：医生ID，请求调整的原班次ID，调整类型（0：调班，1：请假），如果是调班：目标班次时间（日期、时段编号、目标医生ID（可空））（可选），调整原因文本；如果是请假：请假时长（int天数）、调整原因文本

> Body 请求参数

```json
{
  "docId": "string",
  "originalTime": "string",
  "changeType": 0,
  "targetDate": "2019-08-24",
  "timePeriod": 0,
  "targetDoctorId": "string",
  "leaveTimeLength": 0,
  "reason": "string"
}
```

### 请求参数

|名称|位置|类型|必选|说明|
|---|---|---|---|---|
|Authorization|header|string| 是 |认证头，格式: Bearer <token>|
|body|body|object| 否 |none|
|» docId|body|string| 是 |none|
|» originalTime|body|string| 是 |none|
|» changeType|body|integer| 是 |none|
|» targetDate|body|string(date)| 否 |none|
|» timePeriod|body|integer| 否 |none|
|» targetDoctorId|body|string| 否 |none|
|» leaveTimeLength|body|integer| 是 |none|
|» reason|body|string| 否 |none|

> 返回示例

> 200 Response

```json
{
  "code": 0,
  "msg": "string"
}
```

### 返回结果

|状态码|状态码含义|说明|数据模型|
|---|---|---|---|
|200|[OK](https://tools.ietf.org/html/rfc7231#section-6.3.1)|none|Inline|

### 返回数据结构

状态码 **200**

|名称|类型|必选|约束|中文名|说明|
|---|---|---|---|---|---|
|» code|integer|true|none||none|
|» msg|string|true|none||none|
