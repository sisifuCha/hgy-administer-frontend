## GET 根据科室名称获取该科室所有的医生

GET /admin/getDoctorsByDepartment

### 请求参数

|名称|位置|类型|必选|说明|
|---|---|---|---|---|
|departName|query|string| 否 |none|

> 返回示例

> 200 Response

```json
{
  "code": 0,
  "msg": "string",
  "data": [
    {
      "userid": "string",
      "username": "string",
      "department": "string"
    }
  ]
}
```

### 返回结果

|状态码|状态码含义|说明|数据模型|
|---|---|---|---|
|200|[OK](https://tools.ietf.org/html/rfc7231#section-6.3.1)|none|Inline|
