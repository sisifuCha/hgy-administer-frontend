## POST 获取医生列表

POST /admin/getDoctors

> Body 请求参数

```json
{
  "filter_name": "depart",
  "filter_value": "DEP013",
  "page": 1,
  "num": 20
}
```

### 请求参数

|名称|位置|类型|必选|说明|
|---|---|---|---|---|
|Authorization|header|string| 否 |none|
|body|body|object| 是 |none|
|» filter_name|body|string| 是 |限定depart/title/none|
|» filter_value|body|string| 是 |若选定depart作为过滤器则限定为从DEP001到DEP022的id值；若选定title为过滤器则限定为从TITLE001到TITLE004的id值；若选定none为过滤器，则本字段为none|
|» page|body|integer| 是 |第几页|
|» num|body|string| 是 |一页最多多少个医生|
