## POST 医生班次变更申请 

POST /admin/submit-adjustment

提交班次变更申请
输入：请求调整的原班次ID，调整类型（0：调班，1：请假），如果是调班：目标班次时间（日期时间戳YYYY-MM-DD-HH、时段ID），调整原因文本；如果是请假：调整原因文本

> Body 请求参数

```json
{
  "id": "SCH003",
  "doc_id": "DOC0055",
  "changeType": 0,
  "targetDate": "2025-12-25",
  "targetTime": "TIME0001",
  "reason": "enim in laborum qui"
}
```

### 请求参数

|名称|位置|类型|必选|说明|
|---|---|---|---|---|
|Authorization|header|string| 是 |认证头，格式: Bearer <token>|
|body|body|object| 否 |none|
|» id|body|string| 是 |班次（schedule）的id|
|» doc_id|body|string| 是 |医生id|
|» changeType|body|integer| 是 |变更类型，0代表调班，1代表请假|
|» targetDate|body|string| 是 |目标日期|
|» targetTime|body|string| 否 |目标时段，上午是TIME0001，下午是TIME0002|
|» reason|body|string| 否 |调班或请假的原因|

> 返回示例

> 200 Response

```json
{
  "code": 0,
  "msg": "string",
  "data": {}
}
```

### 返回结果

|状态码|状态码含义|说明|数据模型|
|---|---|---|---|
|200|[OK](https://tools.ietf.org/html/rfc7231#section-6.3.1)|none|[res](#schemares)|
