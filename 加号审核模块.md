# 管理端/查看和审批加号

## GET 获取加号请求

GET /admin/getAddedSource

根据单页条目数量和页码获取加号请求详情

### 请求参数

|名称|位置|类型|必选|说明|
|---|---|---|---|---|
|pageSize|query|integer| 否 |分页查询中每一页多少个条目|
|page|query|integer| 否 |分页查询中第几页|
|Authorization|header|string| 否 |none|

> 返回示例

> 200 Response

```json
{
  "code": 0,
  "msg": "string",
  "data": {
    "AddedSourceList": [
      {
        "patient_id": "string",
        "sch_id": "string",
        "apply_time": "string",
        "status": "string",
        "reason_text": "string"
      }
    ]
  }
}
```

### 返回结果

|状态码|状态码含义|说明|数据模型|
|---|---|---|---|
|200|[OK](https://tools.ietf.org/html/rfc7231#section-6.3.1)|none|Inline|

### 返回数据结构

状态码 **200**

|名称|类型|必选|约束|中文名|说明|
|---|---|---|---|---|---|
|» code|integer|true|none||none|
|» msg|string|true|none||none|
|» data|object|false|none||none|
|»» AddedSourceList|[[AddedSource](#schemaaddedsource)]|true|none||none|
|»»» patient_id|string|true|none||none|
|»»» sch_id|string|true|none||none|
|»»» apply_time|string|true|none||none|
|»»» status|string|true|none||none|
|»»» reason_text|string|true|none||none|

## PUT 审批加号请求

PUT /admin/checkAddedSource

根据请求参数修改加号表里对应的status参数，如果status修改为“已同意”，则应同时在挂号表里新增对应的条目

> Body 请求参数

```json
{
  "patient_id": "PAT0027",
  "sch_id": "SCH1006",
  "status": "已同意"
}
```

### 请求参数

|名称|位置|类型|必选|说明|
|---|---|---|---|---|
|Authorization|header|string| 否 |none|
|body|body|object| 是 |none|
|» patient_id|body|string| 是 |none|
|» sch_id|body|string| 是 |none|
|» status|body|string| 是 |none|

> 返回示例

> 200 Response

```json
{
  "code": 0,
  "msg": "string",
  "data": {}
}
```

### 返回结果

|状态码|状态码含义|说明|数据模型|
|---|---|---|---|
|200|[OK](https://tools.ietf.org/html/rfc7231#section-6.3.1)|none|[res](#schemares)|
